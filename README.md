# AWS .NET + React Production Deployment

## Overview

This project demonstrates how to deploy a Dockerized .NET 8 Web API and React application on AWS EC2 using Ubuntu, Nginx, Docker, and Docker Compose.

The repository also documents the equivalent deployment approach for Microsoft Azure and Google Cloud Platform.

## Technologies

- .NET 8
- React
- Docker
- Docker Compose
- Nginx
- PostgreSQL
- AWS EC2
- Azure VM (Reference)
- GCP Compute Engine (Reference)
